視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技術，當網站向下捲動時，背景的移動速度比前景慢。這會產生一種 3D 深度效果，讓使用者在瀏覽網站時有更沉浸式的體驗。視差效果基於視覺錯覺：人眼會將靠近的物體視為較大，而遠處的物體看起來較小，因此我們會感覺遠處物體移動得比較慢。這種錯覺長期以來被運用在不同媒介中，使畫面更具真實感。視差技術最早見於傳統動畫（例如迪士尼的《白雪公主與七個小矮人》）以及早期電子遊戲（如超級瑪利歐）。 隨著 CSS 與 HTML 的進步，視差效果也演變為現代網頁設計的一部分。

最早在瀏覽器中實作視差滾動的範例之一，是由網路開發者 Glutnix 在 2007 年發表的一篇部落格文章，內含示範程式碼與使用 JavaScript/CSS2 的範例，支援當時的瀏覽器（例如 Internet Explorer 6）。然而直到 2011 年 HTML5 與 CSS3 推出後，這項技術才在網頁設計師間廣泛流行。支持者認為視差是一個簡單且能展現網頁流動性的方式。設計師常使用視差背景來提升使用者互動並改善網站整體體驗。但也有研究（例如 2013 年 Purdue University 的研究）指出：「雖然視差滾動在某些面向提升了使用者體驗，但並不見得能改善整體使用者體驗」。


為什麼我應該使用視差圖像？

    設計師採用視差圖像主要有兩個原因：

  • 它能幫助說故事

    視差圖像非常適合用於敘事。好的設計就是良好的溝通——當我們設計網站時，希望能與訪客溝通並說明我們的產品、服務或品牌。視差滾動能把訪客沈浸在獨特的線上體驗中。甚至可以製作單頁網站，讓使用者無需切換頁面即可閱讀完整的品牌或產品故事。

  • 它能提升使用者互動

    視差圖像可用來提升使用者互動。降低網站跳出率是網頁設計中重要的目標之一。跳出率衡量只查看一頁就離開網站的訪客百分比，過高的跳出率會影響搜尋排名，因此任何能降低跳出率的做法都值得嘗試。精心設計的視差頁面通常能提供動態體驗，讓使用者在頁面停留更久。


如果你想在網站上加入視差滾動效果，請考慮以下建議。

  • 選擇性使用

    視差滾動在局部元素上比全站頻繁使用時效果更佳。可將它用在標頭或標題，或僅限於首頁。目標是讓視差強化網站設計，而不是分散使用者注意力。

  • 影像壓縮

    視差滾動通常會使用多種媒體與 CSS 形狀來建立深度感。請避免使用過大的影像檔案，以免影響網站效能與使用者體驗。

  • 玩轉色彩

    視差不僅僅是前景與背景的影像，也關乎色彩。你可以在背景使用較弱的飽和度，並在前景使用較鮮明的色彩，以增強深度感。
